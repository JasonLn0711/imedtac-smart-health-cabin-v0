## 合作案定位

本版將智慧健康量測站合作案主軸調整為租賃優先。若北市聯醫健康量測站經費
來自深耕計畫，採購路徑較可能偏向租賃，報價應先呈現可租、可維護、可升級、
可擴點的產品化方案，而不是以 `NTD 900,000-1,100,000` 的首次建置費作為
對外主版本。

本版依 `2026-07-01` 下午吳老師 LINE 網路電話的修正逐字稿更新；該逐字稿已
將 `Tommy` 修正為 `Tomi`、`小的採購` 修正為 `小額採購`、`編，是` 修正為
`嗯，是`，並將租賃建議來源專名確認為 `余總`。

此版本保留智德萬 / 團隊對軟體、模型、流程、報告、升級與 know-how 的主體
控制，將醫院端定位為使用租賃服務與現場導入。這個說法降低一次性建置費
引發 IP / 原始碼 / 專案成果歸屬爭議的風險，也讓後續院所部署可用租賃與
升級項目延續合作關係。

## 大幅修正摘要

| 修正項目 | v2 方向 | v3 採納方向 |
| --- | --- | --- |
| 對外主軸 | 一套首次建置費 + 後續授權 | 租賃優先 + IP 保留 + 長期升級 |
| 主要價格錨點 | 軟體 / 系統建置 `NTD 900,000-1,100,000` | 軟體租賃參考 `NTD 60,000/year`；小採購友善上限抓 `NTD 140,000-149,000/year` |
| IP 風險 | 大額建置費可能被解讀為院方可 claim IP | 先做成品租賃，軟體/IP/know-how 保留在我方 |
| 客戶接受度 | 首案承擔建置成本，價格壓力高 | 租金貼近硬體/軟體行情，較符合深耕計畫與買方心理 |
| 商業策略 | 首案回收開發成本 | 衝量、留住 customer connection、後續功能加價 |
| Tomi 討論題 | build fee / license / IP | lease model / IP保留 / 小採購門檻 / 加值升級 / 是否另設建置或導入費 |

## 建議報價結構

### A. 軟體租賃主版本

| 項目 | 建議數字 | 說明 |
| --- | ---: | --- |
| 軟體租賃費 | `NTD 60,000/year` | 對應三年 `NTD 180,000`，約 `NTD 5,000/month`。 |
| 維護 / 小幅更新 | 含於租賃或列明基本維護 | 對應賣斷 `NTD 150,000` 加約 `7%` 年維護的思路。 |
| 加值功能 | `NTD 2,000/month` 起另議 | 新功能、進階報表、額外問卷包、語音能力、資料分析等用加值項目承接。 |
| 部署 / 教育訓練 | 另列或首年含一次 | 避免租金被現場導入成本吃掉。 |

### B. 一套設備 + 軟體租賃上限版本

| 項目 | 建議數字 | 說明 |
| --- | ---: | --- |
| 硬體參考 | `NTD 300,000/set` | 仍需慧誠正式硬體價格確認。 |
| 軟體買斷參考 | `NTD 150,000/set` | 作為租賃換算參考，不作為首選對外模式。 |
| 一套參考總價 | `NTD 450,000/set` | 可換算三年租賃約 `NTD 150,000/year`。 |
| 小採購友善租賃 | `NTD 140,000-149,000/year` | 若需要合併設備與軟體租賃，應控制在 `NTD 150,000/year` 以內作為討論區。 |

### C. 保留但不主推的買斷版本

| 項目 | 建議定位 |
| --- | --- |
| 軟體買斷 | 只作備選，不作首選；買斷會降低 customer connection。 |
| 買斷後維護 | 每年約買斷價 `7%-8%`，軟體部分 `NTD 150,000 x 7% ~= NTD 10,500/year`。 |
| IP / 原始碼 | 不隨買斷轉讓；若要求原始碼、IP 或 know-how 讓渡，另案高價啟動。 |

## 對外說法草稿

```text
本案建議採租賃與持續服務方式規劃。智慧健康量測站的軟體、問卷流程、報告、
QR Code、Avatar 介面與後續升級會以既有產品化服務提供，院方可用年度租賃
方式導入，後續如需新增功能、增加問卷包、進階報表或更多場域部署，再依
使用範圍另行加購或升級。
```

## Tomi / 晚上討論要點

1. 租賃主版本是否採 `NTD 60,000/year` 軟體租賃作為對外基本盤。
2. 若要把硬體也包入租賃，是否控制在 `NTD 140,000-149,000/year`。
3. `NTD 900,000-1,100,000` 是否只保留為內部開發成本 / 客製專案參考，不再作
   對外主報價。
4. 對外文字如何明確保留軟體 IP、模型、流程、know-how、原始碼與升級主控權。
5. 哪些功能列入基本租賃，哪些用每月加價、年度加價、或另案客製承接。
6. 若醫院或慧誠要求買斷、原始碼、資料使用權、HIS/PACS 整合、正式醫材驗證，
   是否全部列為另案啟動。
