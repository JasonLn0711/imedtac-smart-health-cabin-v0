# 慧誠智醫 Smart Health Cabin 吳老師內部報價單

## 文件定位

本報價單供吳老師與 NYCU 團隊內部討論 Smart Health Cabin 合作案的報價
策略、範圍切分與談判底線。這份文件不是對慧誠智醫或院方的正式簽核報價。

本案目前應以「開放式健康量測站」為主體，報價要把硬體、Avatar vendor
費用、AI mini PC、HIS 正式串接與 NYCU 軟體開發費用分開。若業主要求
`NTD 1,500,000` 同時包含兩台硬體與軟體，NYCU 軟體 scope 必須同步縮小。

## 報價結論

兩台全包 `NTD 1,500,000` 可以報，但不能稱為「完整四模組版」；應稱為：

```text
智慧健康量測站兩台部署暨 NYCU 四模組 MVP 精簡版
```

理由很直接：若 2 台健康量測站硬體先以 `NTD 300,000/台` 暫估，硬體合計
`NTD 600,000`，總預算 `NTD 1,500,000` 中只剩 `NTD 900,000` 可用於
NYCU 軟體。這低於四模組底線版 `NTD 1,060,000`，因此視力、聽力必須降為
保守自我篩檢，Avatar 限定為既有 vendor 介面整合，且不包含 Avatar
license、AI mini PC、HIS 正式串接或長期維運。

## 三層價格依據

### 1. 資訊服務委外人月法

中華民國資訊軟體服務商業同業公會 115 年資訊服務委外經費估算原則可作為
資訊服務報價的人月依據。第一類資訊服務案可引用的人月參考價包含：

| 職能 | 參考人月單價 |
| --- | ---: |
| 專案管理師 | `NTD 223,083` |
| 系統分析設計師 | `NTD 203,337` |
| 軟體開發人員 | `NTD 181,502` |
| AI 分析人員 | `NTD 216,431` |
| 雲地整合人員 | `NTD 226,912` |
| 測試人員 | `NTD 163,261` |

本案使用人月反推法：`NTD 900,000` 約只能支撐 `4.5-5` 人月，因此只能是
MVP 精簡交付；`NTD 1,060,000` 約支撐 `5.5` 人月，可作為四模組底線版。

來源：<https://www.tissa.org.tw/News/Detail/6123>

### 2. 公開標案類比法

公開標案不是本案同案，但可以作為醫療場域資訊系統、IoT 整合、報告/資料
流程與驗收支援的行情參考。

| 類比標案 | 決標金額 | 可參考意義 |
| --- | ---: | --- |
| 高雄市立民生醫院「健檢資訊系統建置、轉換及租用服務」 | `NTD 950,100` | 健檢資訊系統建置、轉換、教育訓練與後續資料處理服務。 |
| 仁德醫院 115 年「智慧科技病房 IoT 整合建置租賃服務」 | `NTD 1,499,000` | 醫療場域 IoT 整合與租賃服務。 |
| 新北市立聯合醫院「護理資訊及護理站電子白板系統建置案」 | `NTD 4,400,000` | 護理資訊與站台電子白板系統建置。 |

這些標案支持一個談判重點：百萬級醫療資訊整合報價是正常行情，不是浮報。

參考：<https://bid.twincn.com/item.aspx?sn=16606810648>

### 3. 健康量測站硬體類比法

健康量測站不是單純平板或電腦。市面智慧健康量測站通常整合連網硬體、軟體
平台、APP 與生理量測設備，可涵蓋血壓、血氧、血糖、額溫、體重、體脂等
量測項目。

因此 `NTD 300,000/台` 只能作為低估情境。若量測站包含醫療級設備、感應卡
讀取、雲端同步、會員管理、量表管理、生理量測管理、影音衛教、預約或遠距
通訊，硬體價格可能大幅上升。

參考：<https://tw.shuttle.com/products/productsDetail?c=medical-healthcare&pn=KHS+SERIES>

### A-5. 對業主說法

本案若採兩台健康量測站全包方式，並以每台硬體暫估新臺幣 `300,000` 元
計算，兩台硬體合計為新臺幣 `600,000` 元，總預算新臺幣 `1,500,000` 元
中可分配至 NYCU 軟體開發與系統整合之預算為新臺幣 `900,000` 元。

此版本可提供四個模組之 MVP 精簡交付：問卷模組、Avatar 導引整合、報告與
QR Code 為第一階段核心；視力與聽力模組則以保守自我篩檢、結果提示與報告
整合為範圍，不宣稱醫療診斷、正式醫材驗證或取代專業人員判斷。

依 115 年資訊服務委外經費估算原則，軟體開發、系統分析、AI 整合、雲地
整合、測試與資安檢查皆有明確人月成本參考。以本案新臺幣 `900,000` 元軟體
預算反推，約可支撐 `4.5-5` 人月之有效投入。因此，本版本定位為兩台部署與
四模組 MVP 精簡版，而非完整產品化版本。

## 正式報價前待確認

1. 這次是一台或兩台健康量測站。
2. `NTD 1,500,000` 是硬體預算、總案預算，還是內部情境數字。
3. 慧誠是否能提供正式硬體報價。
4. Avatar vendor 是 API、iframe、web component、SDK、cloud service，還是完整前台方案。
5. Avatar vendor license / API 使用費由誰負責。
6. AI mini PC / GPU box 是否需要採購，或是否可走 vendor-hosted / cloud path。
7. 是否需要 HIS / EMR 正式串接；若需要，資料格式、院方 IT、資安審查與驗收另計。
8. 是否包含驗收後維運。
9. 是否要求原始碼交付、完整讓渡或永久商用授權。
