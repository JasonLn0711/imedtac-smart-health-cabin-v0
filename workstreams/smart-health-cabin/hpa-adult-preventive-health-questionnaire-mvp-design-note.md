# HPA Adult Preventive Health Questionnaire MVP Design Note

This note will become the questionnaire-module MVP design record after the HPA
adult preventive health source package and any WHO STEPS references are reviewed.

## Current Scope Control

The questionnaire module should support public-health self-assessment and
staff-review workflow. It should not become autonomous diagnosis, treatment
advice, or unreviewed medical triage.

## Pending Inputs

- HPA adult preventive health form source.
- WHO STEPS reference fields.
- imedtac / hospital owner confirmation.
- Report wording owner.
- Versioning and review workflow.

## MVP Output Target

- structured answers;
- questionnaire version;
- rule / scoring trace when used;
- report-ready summary;
- staff-review prompt;
- activation state per customer.
